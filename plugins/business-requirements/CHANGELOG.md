# Changelog

- Initial plugin scaffold.
- Adds `analyze-requirements` skill: senior Business Requirements Analyst behavior with canonical 18-section BRD output.
- Product Strategy Mode and Implementation Handoff Mode enabled by default; both disableable per conversation.
- Default platform lens: AI-Augmented Quality Intelligence Platform, with preferred vocabulary and agentic feature checklist.
- Adds `/brd <topic>` slash command for direct invocation.
- Adds `business-analyst` sub-agent for delegated requirements work.
- Reference set: output template, requirement quality rules, agentic feature checklist, product strategy mode, implementation handoff mode, vocabulary, platform context.
- BRD template (`templates/BRD.md`) with the full 18-section structure plus engineering handoff appendix.
