# Changelog

## 0.1.0 — 2026-05-21

- Initial plugin scaffold.
- Adds `exploratory-to-bdd` skill: convert exploration notes, Playwright MCP transcripts, user stories, acceptance criteria, manual test cases, and bug reproductions into reviewable BDD artifacts.
- Slash commands: `/explore-to-bdd`, `/generate-bdd`, `/review-bdd`, `/execute-bdd-mcp`.
- Templates: Markdown BDD spec, Gherkin feature, traceability matrix, BDD quality review, automation candidate.
- Checklists: BDD quality, ambiguity and defect, automation candidate.
- SauceDemo examples: login Markdown spec, login `.feature`, traceability matrix, BDD quality review.
